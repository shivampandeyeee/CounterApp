## Description
Counter App is a simple Android application developed using **Java** in **Android Studio**.  
The app allows users to increase, decrease, and reset a counter value.  
This project is created as part of my learning process to understand basic Android development concepts.

## Features
- Increment counter value  
- Decrement counter value  
- Reset counter to zero  
- Simple and user-friendly UI  

## Tech Stack
- Java  
- Android Studio  
- XML (UI Design)  

## Learning Outcomes
- Understanding Android activity lifecycle  
- Handling button click events  
- Designing UI using XML  
- Basic app structure in Android Studio
